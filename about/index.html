This is whatyousayiswhatyouget.net, the master’s thesis of Fabian Pitzer. You can find more of his work at pitzer.xyz. He looked into how graphic designers can build and use custom tools deliberately, making the most out of computers while keeping sustainability in mind. He links this to the language beneficial to instructing large language models, which are good at generating code.

On the Dictionary page, you will find a graph that attempts to map some of his favorite graphic design works to the computational language underlying its systems. The taxonomy is more entangled in reality but should serve as an overview for what kind of tools can be built. The projects showcase techniques that are suitable for reaping the benefits of custom tools.

The Tools page hosts implementations of some of the algorithms mentioned in the dictionary. Exposing part of the code intends to demystify it. If you want to change the logic of the program or modify it to use e.g. a different font, you can download the source code from GitHub.

The Manifesto aims to critique the lavish use of LLMs to generate code based on vibes and urges a more considered approach.

The Thesis page hosts the theoretical part of this work and is set in Times New Roman and True Sans Mono by Mads Wildgaard.
